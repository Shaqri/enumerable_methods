# Enumerable Methods

> In this project we replicate the functionality of the following enumerable methods: 
- #my_each
- #my_each_with_index
- #my_select
- #my_all?
- #my_any?
- #my_none?
- #my_count
- #my_map
- #my_inject
- #multiply_els
